Overview

Main entry point for the game. This class manages the current scene which can be changed with the `#go_to_scene` method. It delegates the main events of `Gosu::Window`, such as drawing, updating and handling keystrokes, to the current scene.

By default it starts with the scene `Bakeru::Scenes::MainMenu`

def button_down(id)
  if id == Gosu::KB_ESCAPE
    go_to_scene(Scenes::MainMenu)
  else
    current_scene.button_down(id)
  end
end

def go_to_scene(scene_class, options={})
  previous_scene = current_scene
  @current_scene = setup_scene(scene_class, options)
  previous_scene.teardown
end
